# SEO Strategy

## In scope
- The browser-facing public web entry point served by the Express + Expo web deployment
- Any static or prerendered marketing/pricing content emitted by the web build
- Crawlability assets for the deployed web experience, including `robots.txt`, `sitemap.xml`, and web metadata

## Out of scope
- Native App Store / Play Store ASO
- In-app authenticated or task-oriented flows after the SPA has booted
- JSON API endpoints under `/api/**`

## Rendering model
- Expo React Native Web export served as a client-rendered SPA via Express fallback routing
- No source-controlled SSR/SSG marketing pages were found in the repo

## Target audience
- Arabic-speaking car owners in Saudi Arabia looking for spare-part quotes
- Spare-parts vendors and workshops receiving RFQs

- طلب عرض سعر قطع غيار
